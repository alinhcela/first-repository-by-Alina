# Порядок виконання роботи:

## 1. Відкрила git через термінал
## 2. Створила .gitignore та проігнорувала вказані файли
## 3. Додала в локальне сховище всі файли - git add .
## 4. Створила коміт (тим самим зберегла головну гілку до змін) - git commit -m "main commit"
## 5. Створила нову гілку "new-branch" - git branch new-branch
## 6. Створила коміт для збереження нової гілки - git commit -m "new branch"
## 7. У new-branch створила папку blog з файлами index.js, index.html - git mkdir blog -> git eche >> index.js -> git eche >> index.html 
## 8. Завантажила гілку на репозиторій - git push -u origin master
